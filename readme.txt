1. There are two files one is .jmx - Jmeter file and one is .csv file - Parameter file.
2. Copy these 2  files from git hub to local jmeter/bin folder.
3. open the .jmx file.
4. Currently the .jmx thread group is configured to execute for 2 concurrent users and for 60 sec of duration
5. View Results Tree and Aggregate Report Listener has been added.
6. Click on the green button on Jmeter tool and the test will be executed for total 62 sec- 2 sec delay has been added.
7. you should be able to view the test results in Aggregate Report secion.
